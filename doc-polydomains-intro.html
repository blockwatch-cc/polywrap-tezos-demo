Polydomains

Introduction

Polydomains is a Tezos domains Polywrap integration project that allows you to search for registered Tezos domain names across Tezos testnets and mainnet. The network to search on is detected from the TLD domain you add (.tez for mainnet and .gra or .han for one of the testnets).

  • Visit the demo app here.
  • Github repository for the app is here.

GitHub Repo & Project overview

The Polydomains wrapper and example can be found in the Poly Repo. In the folder polydomains there's the wrapper folder which contains the codes for the polydomains wrapper and example folder folder which contains a react implementation of the polydomains wrapper.
