Hic Et Nunc (HEN)

Introduction

Hic Et Nunc (HEN) is an experimental open-source NFT platform that serves as ''a public smart contract infrastructure''. Hic Et Nunc Wrapper aims to to make functions call to the Hic Et Nunc API available in a polywrapper.

  • Visit the demo app here.
  • Github repository for the app is here.

GitHub Repo & Project overview

The Hic Et Nunc (HEN) wrapper and example can be found in the Poly Repo. In the folder Hicetnunc there's the wrapper folder which contains the codes for the Harbinger wrapper and example folder folder which contains a React JS implementation of the Hicetnunc wrapper.
